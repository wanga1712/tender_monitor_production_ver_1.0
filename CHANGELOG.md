# Changelog

Все значительные изменения в проекте TenderMonitor документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено
- Файл `requirements.txt` с зависимостями проекта
- Файл `ROADMAP.md` с дорожной картой проекта
- Файл `IMPROVEMENTS.md` с предложениями по улучшению
- Файл `AUDIT_REPORT.md` с отчетом об аудите кода
- Файл `.gitignore` для исключения ненужных файлов из репозитория
- Файл `CHANGELOG.md` для отслеживания изменений
- Обновлен `README.md` с полной документацией

### Исправлено
- Критическая ошибка в `eis_requester.py`: исправлена вложенность циклов обработки подсистем 223-ФЗ
- Ошибка в `generate_soap_request`: добавлена правильная логика выбора поля documentType44/documentType223
- Ошибка в загрузке конфигурации: исправлено несоответствие имен ключей в config.ini
- Ошибка в `okpd_parser.py`: исправлена инициализация FileDeleter с правильным путем
- Hardcoded путь к БД: путь теперь определяется относительно текущего модуля
- Улучшена обработка ошибок при закрытии соединений с БД
- Улучшено управление ресурсами (курсоры и соединения с БД)

### Удалено
- Резервный файл `utils.py~`
- Файл `1234_delete.py` (не относится к проекту)

### Изменено
- Улучшена структура кода для лучшей читаемости
- Улучшена документация в коде
- Оптимизирована задержка в `file_downloader.py` (с 5 до 2 секунд)

## [1.0.0] - 2025-01-27

### Добавлено
- Базовая функциональность мониторинга тендеров
- Поддержка 44-ФЗ и 223-ФЗ
- Парсинг XML-файлов
- Работа с базой данных PostgreSQL
- Фильтрация по ОКПД кодам
- Логирование операций
- Поддержка stunnel для безопасного подключения

## Формат версий

- **MAJOR** версия - несовместимые изменения в API
- **MINOR** версия - новая функциональность с обратной совместимостью
- **PATCH** версия - исправления ошибок с обратной совместимостью

## Типы изменений

- **Добавлено** - новые функции
- **Изменено** - изменения в существующей функциональности
- **Устарело** - функции, которые скоро будут удалены
- **Удалено** - удаленные функции
- **Исправлено** - исправления ошибок
- **Безопасность** - исправления уязвимостей
